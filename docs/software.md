## Software previo recomendable

- Lenguaje **[Python](http://python.org/)**: Una versión de sistema de
  Python 3.9 o superior.  El proyecto se desarrollo sobre Python 3.12.3,
  si quieres ir a lo seguro.

- La utilidad **[uv](https://docs.astral.sh/uv/): Una herramienta
  única que intenta reemplazar a distintas utilidades del
  ecosistema Python: `pip`, `pip-tools`, `pipx`, `poetry`, `pyenv`,
  `twine` y `virtualenv` entre otras.

- La utilidad **[just](https://just.systems/man/en/), un ejecutor de
  comandos escrito en [rust](https://rust-lang.org/)
